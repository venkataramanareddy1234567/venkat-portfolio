This is my first personal portfolio website, built using HTML, CSS, and JavaScript.
The purpose of this project is to showcase my profile, projects, certificates, and contact details in a simple and professional way.

I created this project to practice front-end fundamentals and to understand how a real website is structured and deployed.

Tools & Technologies Used
-HTML
-CSS
-JavaScript

Features:
-Responsive personal portfolio layout
-Animated background and smooth UI
-Projects section with live project and GitHub links
-Certificates section with verification links
-Contact section with email and social links
-Mobile-friendly design
